## 🎯 Key Design Decisions
- **Vanilla Tech Stack**: Chosen for zero-dependency baseline performance.
- **CSS Custom Properties**: Ensures scalable theming and easy Dark/Light mode toggling.
- **Intersection Observers**: Used exclusively for scroll-reveal animations to keep CPU usage low.
- **Glassmorphism**: Achieved with `backdrop-filter: blur` and translucent backgrounds for a premium feel.
